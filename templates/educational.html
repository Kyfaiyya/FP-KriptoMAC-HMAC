Learn MAC & HMAC

What is MAC?

Message Authentication Code (MAC) adalah nilai kriptografi yang digunakan untuk memverifikasi integritas dan autentisitas pesan.

Fungsi Utama MAC:
  • Authentication: Memverifikasi bahwa pesan berasal dari pengirim yang sah
  • Integrity: Memastikan pesan tidak diubah selama transmisi
  • Non-repudiation: Pengirim tidak dapat menyangkal telah mengirim pesan
Cara Kerja MAC:
  1. Pengirim menghitung MAC dari pesan menggunakan secret key
  2. MAC dikirim bersama dengan pesan
  3. Penerima menghitung ulang MAC menggunakan secret key yang sama
  4. Jika MAC yang dihitung sama dengan MAC yang diterima, pesan valid

What is HMAC?

Hash-based Message Authentication Code (HMAC) adalah implementasi MAC yang menggunakan fungsi hash kriptografi.

Formula HMAC:
HMAC(K, m) = H((K ⊕ opad) || H((K ⊕ ipad) || m))

Dimana:

  • K: Secret key
  • m: Message
  • H: Hash function (SHA-256, SHA-512, dll)
  • opad: Outer padding (0x5C berulang)
  • ipad: Inner padding (0x36 berulang)
  • ⊕: XOR operation
  • ||: Concatenation
Keunggulan HMAC:
  • Tahan terhadap length extension attacks
  • Menggunakan fungsi hash yang sudah teruji
  • Standar industri (RFC 2104)
  • Efisien dan mudah diimplementasi

Implementation Details

1. Simple MAC

Implementasi sederhana: MAC = H(key || message)

Kelemahan: Rentan terhadap length extension attacks pada beberapa hash function.
2. Improved MAC

Implementasi yang lebih baik: MAC = H(key || H(message))

Keunggulan: Lebih tahan terhadap beberapa jenis serangan.
3. HMAC (Recommended)

Implementasi standar yang menggunakan nested hash dengan padding.

Rekomendasi: Gunakan HMAC untuk aplikasi production.
Security Considerations
Key Management:
  • Gunakan key yang cukup panjang (minimal 128 bit)
  • Key harus random dan unpredictable
  • Jangan hardcode key dalam source code
  • Rotasi key secara berkala
Algorithm Choice:
  • Recommended: SHA-256, SHA-512
  • Deprecated: SHA-1
  • Broken: MD5
Use Cases
  • API Authentication: Verifikasi request API
  • File Integrity: Deteksi perubahan file
  • Digital Signatures: Komponen dalam skema tanda tangan
  • Secure Cookies: Proteksi session cookies
  • Database Integrity: Verifikasi data database
Tools & Libraries
Python:
  • hashlib - Built-in hash functions
  • hmac - Built-in HMAC implementation
  • cryptography - Advanced crypto library
Other Languages:
  • JavaScript: crypto module, Web Crypto API
  • Java: javax.crypto.Mac
  • C#: System.Security.Cryptography
